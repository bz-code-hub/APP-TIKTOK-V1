# TikTok Live Stream Simulator

A modern web application that simulates a TikTok Live streaming experience.

## Setup

```bash
npm install
npm run dev
```

## Configuration

Edit `src/config/livestream-config.ts` to customize:
- Video type (YouTube, Vimeo, Panda, VTurb, or Direct)
- Channel name and profile image
- Viewer counts and timing
- Chat messages
- CTA button

## Build for Production

```bash
npm run build
```

Output will be in the `dist/` folder.
